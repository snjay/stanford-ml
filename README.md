### FAQs
If you encounter the following error whilst running on MacOS
```
line 0: unknown or ambiguous terminal type; type just 'set terminal' for a list
```

Run the following command:
```
$ echo "setenv(\"GNUTERM\",\"qt\")" > ~/.octaverc
```
